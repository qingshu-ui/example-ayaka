## 安装和使用

### 克隆仓库

```bash
git clone https://github.com/qingshu-ui/example-ayaka.git
cd example-ayaka
```

### 模型下载

> https://drive.google.com/drive/folders/16O7cpg8oXEB4W8Rixq-MoSrY9AMhZOsj?usp=sharing
